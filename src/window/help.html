how 2 use jside

there's a left side and a right side. the left side is a code editor, the right side is a console

if you drag a bar all the way to one side, it will autohide. dont worry, its still there

press ctrl + enter to run the code you wrote. you need a return or something with console if you want something to show

you can use some built in nodejs modules, such as fs or http. enable them in the permissions menu.

your code automatically backs up to your appdata folder, so dont worry if you close it by accident

here are some keyboard shortcuts if u want:

  • ctrl + s save file
  • ctrl + o open file
  • ctrl + alt + s format code
  • ctrl + l clear console

explore the menu for all of them
